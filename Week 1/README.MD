🌸 Blooming Petals Flower Shop - Website
A clean, responsive static website for Blooming Petals Flower Shop, showcasing fresh floral arrangements, business information, and a link to the full site. This HTML page is designed to introduce visitors to the brand with a focus on quality, sustainability, and local sourcing.

📄 Overview
This project is a standalone HTML file (index.html) that serves as the homepage for a flower shop. It includes:

SEO-friendly metadata
A welcoming header with navigation
An "About Us" section highlighting the shop’s values and history
Business hours and delivery information
Footer with copyright notice
The site emphasizes:

Fresh, locally-sourced flowers
Sustainable practices
Hand-crafted floral arrangements
Same-day delivery
🚀 Features
✅ SEO Optimized
Includes meta tags for description, keywords, author, and robots to improve search engine visibility.

✅ Mobile Responsive (via viewport meta tag)
Designed to scale appropriately on mobile devices.

✅ Accessibility & Semantics
Uses proper HTML5 semantic tags (<header>, <section>, <nav>, <footer>).

✅ Sustainability Focus
Highlights eco-friendly practices like composting, recycling, and local sourcing.
